Welcome to MegaApiClient's documentation!
=========================================

MegaApiClient is a .Net library to access `http://mega.co.nz <http://mega.co.nz>`_ / `http://mega.nz <http://mega.nz>`_ cloud storage and file hosting service.

The library is fully managed and supports .NET Framework >=4.0 / `.NET standard >= 1.3 <https://docs.microsoft.com/en-us/dotnet/standard/net-standard>`_ (.NET Core).
NewtonSoft.Json >=9.0.1 is the only required dependency.
